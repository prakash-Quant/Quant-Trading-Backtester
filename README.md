# Simple Moving Average (SMA) Backtester
This project analyzes stock market data using Python to test a "Trend-Following" strategy.

## Features
* Downloads real-time data using `yfinance`.
* Calculates 20, 50, and 200-day Moving Averages.
* Includes a trend filter to reduce risk.
* Calculates the **Sharpe Ratio** and **Cumulative Returns**.

## Libraries Used
* Pandas
* NumPy
* Matplotlib
* yFinance
